## 📌 Project Overview

The **Student Interaction Portal** is a menu-driven Python application built using core programming concepts such as:

- Functions
- Conditional Statements
- Loops
- Lists
- Tuples
- Dictionaries
- File Handling

The portal serves as a centralized platform where students can:

- Create and manage accounts
- Recover forgotten passwords
- View academic information
- Explore student clubs
- Join campus clubs
- View upcoming events
- Chat with seniors
- Create and edit profiles
- Submit feedback
- Access campus rules and regulations

It also includes an **Admin Dashboard** for managing users and viewing feedback.
